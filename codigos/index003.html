ESTRUTURA DE UM HTML

O HTML precisa de uma tag de abertura, um conteúdo e um uma tag de fechamento, algumas não possui tag de fechamento.

BASIC HTML

HTML <!DOCTYPE> DECLARATION

Defines the document type.

<!DOCTYPE html>

HTML <html> TAG

Defines an HTML document.

<html lang=" "> </html>

HTML <head> TAG

Contains metadata/information for the document.

<head> </head>

HTML <title> TAG

Defines a title for the document.

<title> </title>

HTML <body> TAG

Defines the document's body.

<body> </body>

HTML <h1> to <h6> TAGS

Defines a title for HTML headings.

<h1> </h1> ... <h6> </h6>

HTML <p> TAG

Defines a paragraph.

<p> </p>

HTML <br> TAG

Inserts a single line break.

<br>

HTML <hr> TAG

Defines a thematic change in the content.

<hr>

HTML <!-- ... > Tag

Defines a comment.

<!-- ... >
