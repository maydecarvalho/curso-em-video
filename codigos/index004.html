ESTRUTURA DE UM CSS

O CSS precisa de um seletor, uma chave de abertura, uma declaração com uma propriedade e valor, uma chave de fechamento.

CSS3

* = Configuração Globais CSS

Shorthand -> Propriedades com vários valores

animation: _____;

background: _____; = (color -> image -> repeat -> size -> attachment) obs: o size n funciona na shorthand no momento.

border: ___; = (border-width -> border-style -> border-color)

border-radius: ___; = Arredondar a borda (px)

box-shadow: ___; = (eixo X (+ ou -) -> eixo Y (+ ou -) -> desfoque -> expandir -> cor -> inset = criar a sombra na frente do elemento)
(X + = sombra direita, X - = sombra esquerda, Y + = sombra pra baixo, Y - = sombra pra cima)

font: ___; = (font-style -> font-weight -> font-size -> font-family)

outline: ___; = (outline-width -> outline-style -> outline-color)

Pseudo Classes

:active = Aplicar estilo quando clicar e segurar um link

:hover = Aplicar um estilo quando passar o mousse em cima

:visited = Aplicar um estilo quando clicar no link

background-image: linear-gradient(_____); = Cor de Fundo Gradiente

background-image: url(_____);

background-size: _____;

background-repeat: _____;

background-position: _____;

text-indent: ___; px = Indentação no Início do Parágrafo

border-radius -> Arredondar a Borda

border-top-left-radius: ___; = Em cima e esquerda (px)

border-top-right-radius: ___; = Em cima e direita (px)

border-bottom-right-radius: ___; Em baixo e direita (px)

border-bottom-left-radius: ___; = Em baixo e esquerda (px)

border-spacing: ___; = Tira o espaço entre as bordas da tabela

font-weight: ___; = Peso (lighter, normal, bold, bolder) ou (100 - 900)

Outline -> Espaço entre a borda e a margem

outline-color: ___; = Cor (nominal, hexadecimal, rgb, hsl)

outline-style: ___; = Estilo (dotted -> Pontilhado, dashed -> Traçado, solid -> Linha, double -> Linha dupla, groove -> Linha 3D)

outline-width: ___; = Tamanho (px)
