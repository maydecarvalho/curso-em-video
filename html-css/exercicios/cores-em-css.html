CORES EM CSS

Este título está colorido usando cores por nome (background-color: hotpink)

Já esse está usando cores pelo código Hexadecimal (background-color: #ff69b4;)

Este outro está colorido usando códigos RGB (red, green, blue) (background-color: rgb(255, 105, 180))

Este outro aqui está usando códigos HSL (hue, saturation, light) (background-color: hsl(330, 100%, 71%))

Também é possível usar transparência, usando o formato RGBA (red, green, blue, alpha) (background-color: rgba(255, 105, 180, 0.5))
